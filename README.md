# Blog Project

This project is a full-stack blogging platform developed as part of an online Full Stack Development course offered by my college. It allows users to create, read, update, and delete blog posts with authentication features.

## Features
- User authentication (signup/login/logout)
- Create, edit, and delete blog posts
- View all blog posts
- Responsive design for different devices

## Technologies Used
- **Frontend:** HTML, CSS, JavaScript, React.js
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Authentication:** JWT (JSON Web Token)
- **Styling:** Bootstrap

## Installation and Setup
To run the project locally, follow these steps:

1. Clone the repository:
   ```bash
   git remote add origin https://github.com/pranathi-vnr/Blog_Project
   git pull origin master
2.Run in terminal:
  cd client
  npm install
  npm run dev
  
## Usage
Register or log in to create and manage blog posts.
Browse and read existing blogs.
Edit or delete blogs created by the logged-in user.
